# Insertion Sort List

Sort a linked list using insertion sort.

![alt text](https://upload.wikimedia.org/wikipedia/commons/0/0f/Insertion-sort-example-300px.gif)


Example 1:

    Input: 4->2->1->3
    Output: 1->2->3->4
    

Example 2:

    Input: -1->5->3->4->0
    Output: -1->0->3->4->5
